# 日志查看器 v2.0 - 使用说明

## ✅ 项目状态

**后端服务已成功运行！** 🎉

- FastAPI服务：http://127.0.0.1:8000
- 演示日志已自动加载
- 可在浏览器中直接使用

## 🚀 快速开始

### 方式1：浏览器访问（推荐）

直接打开浏览器访问：**http://127.0.0.1:8000**

### 方式2：使用启动脚本

双击运行：`start.bat`

## 📖 功能说明

### 1. 查看演示日志
- 打开浏览器访问 http://127.0.0.1:8000
- 页面会自动加载 `test_trace_simple.md` 演示文件
- 显示在第一个标签页中

### 2. 上传新日志
- 点击页面右上角的 "+" 按钮
- 选择 `.md` 日志文件
- 文件会自动解析并在新标签页中显示

### 3. 标签页管理
- **切换标签**：点击标签页头部
- **关闭标签**：点击标签页的 "×" 按钮
- **多文件对比**：可同时打开多个日志文件
